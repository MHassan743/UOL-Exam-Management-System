Welcome to UOL CS Exam Schedule

How to Use This System:

1
Upload Your Excel File

Click the upload area in the sidebar or drag & drop your Excel file (.xlsx, .xls, .csv). The system will automatically filter CS department data.

2
Filter by Program/Semester

Use the program filters in the sidebar to view specific programs (BSCS, BSSE, etc.), semesters, or sections. Click "Clear All" to reset filters.

3
View and Search Data

View the schedule in table format. Use the search box to find specific entries. Click any row to see detailed exam information.

4
Export Data

Use the action buttons to export data as Excel or PDF files, print the schedule, or refresh the data.

5
Theme & Display

Toggle between light and dark themes using the theme button in the top navigation bar.

Important Notes:
  • Only Excel files with CS department data will be processed
  • Date columns should be properly formatted in Excel
  • Program names should include CS codes (e.g., BSCS-6, BSSE-5A)
  • Supported formats: .xlsx, .xls, .csv
